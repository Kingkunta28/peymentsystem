# Frontend Setup (React + Vite)

## Run

```bash
cd frontend
npm.cmd install
npm.cmd run dev
```

Frontend URL: `http://127.0.0.1:5173`

## API Base URL

Default API base is:

`http://127.0.0.1:8000/api`

Override with `.env` in `frontend/`:

```env
VITE_API_BASE=http://127.0.0.1:8000/api
```

## Build

```bash
npm.cmd run build
```
